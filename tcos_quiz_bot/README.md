
# Telegram-бот "ТЦОС Квиз"

Данный бот предназначен для тестирования студентов по дисциплине **"Теория Цифровой Обработки Сигналов" (ТЦОС)**. Бот поддерживает несколько тем, вопросы с вариантами ответов и изображениями.

## Технологический стек
- Python 3.9+
- [python-telegram-bot v20.x](https://github.com/python-telegram-bot/python-telegram-bot) (асинхронная библиотека)
- `python-dotenv` для управления секретами
- `Pillow` для обработки изображений (если потребуется расширение функционала)

## Установка и Запуск

### 1. Подготовка
Убедитесь, что у вас установлен Python версии 3.9 или выше.

### 2. Клонирование и настройка
Склонируйте файлы проекта в рабочую папку.

```bash
cd tcos_quiz_bot
```

### 3. Виртуальное окружение
Создайте и активируйте виртуальное окружение:
```bash
python -m venv venv
# Для Windows:
venv\Scripts\activate
# Для Linux/macOS:
source venv/bin/activate
```

### 4. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 5. Получение токена
1. Напишите [@BotFather](https://t.me/botfather) в Telegram.
2. Создайте нового бота с помощью команды `/newbot`.
3. Скопируйте полученный API Token.

### 6. Переменные окружения
Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```
Откройте `.env` и вставьте ваш токен в переменную `TELEGRAM_BOT_TOKEN`.

### 7. Запуск
```bash
python bot.py
```

## Добавление вопросов

Все вопросы хранятся в файле `quiz_data.py` в виде словаря `QUIZ_DATA`.

### Формат вопроса:
```python
{
    "question": "Текст вашего вопроса?",
    "type": "multiple_choice", # или "binary"
    "options": ["Вариант 1", "Вариант 2", "Вариант 3"], # Список вариантов
    "answer": "Вариант 1", # Строка, в точности совпадающая с правильным вариантом
    "image": "https://url.to/image.png" # Или None, если картинка не нужна
}
```

Чтобы добавить новую тему, просто добавьте новый ключ в словарь `QUIZ_DATA` и список вопросов к нему.

## Структура проекта
- `bot.py`: Основная логика, обработчики команд и состояний.
- `config.py`: Загрузка настроек из окружения.
- `quiz_data.py`: База данных вопросов.
- `requirements.txt`: Зависимости проекта.
- `.env.example`: Шаблон для конфиденциальных данных.
